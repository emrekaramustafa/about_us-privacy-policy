# Göz Testi / Eye Test - Web Pages

This directory contains the web pages for the Göz Testi / Eye Test mobile application.

## Pages

- **index.html** - Main landing page
- **privacy-policy.html** - Privacy Policy
- **contact.html** - Contact Us page
- **terms-of-service.html** - Terms of Service
- **app-ads.txt** - AdMob publisher verification (Eye Test iOS); live at `/app-ads.txt`

## GitHub Pages Setup

To enable GitHub Pages:

1. Go to your repository settings on GitHub
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/docs" folder
5. Click "Save"

Your pages will be available at:
`https://emrekaramustafa.github.io/goz-testi/`
